== Description ==

Plugins for modify some word from text, this plugins created due to some mod_security rule that not allow any special word in POST/GET Action 

== Installation ==

This section how to install the plugin and get it working.

e.g.

1. Upload the folder wwmodifword to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. No need to place any tag, this will work after Publish / Update button pressed in Post Editor
